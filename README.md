
# Social Media Downloader

Social Media Downloader is a tool that allows you to download media from popular social media platforms, including YouTube, Facebook, Instagram, and TikTok. Built for simplicity, this tool supports single and batch downloads and includes automatic update checking.

## Features
- **Platform Support**: Easily download videos from YouTube, TikTok, Instagram, and Facebook.
- **Format Selection**: Choose from multiple video formats and quality options.
- **Batch Download**: Download multiple URLs at once by specifying a list of URLs in a text file.
- **Update Check**: Automatically check for new versions and apply updates if desired.
- **Logging**: Each download is logged with a timestamp and status, stored in `download_history.csv`.

## Usage
This tool is designed with a user-friendly command-line interface. Upon running, select the platform and input the media URL, or choose batch download for bulk processing.

### Platform Options
1. **YouTube Download**: Download YouTube videos with format selection.
2. **Facebook Download**: Fetch and download Facebook videos.
3. **Instagram Download**: Download individual posts from Instagram.
4. **TikTok Download**: Retrieve videos from TikTok.
5. **Batch Download**: Enter a file containing multiple URLs for bulk downloads.
6. **Check for Updates**: Automatically checks and updates to the latest version if available.
7. **Help**: Displays usage instructions and features.
8. **Exit**: Close the downloader.

### Downloading and Installation
1. [Download the latest executable version here](https://github.com/nayandas69/social-media-downloader/releases/latest) to start using Social Media Downloader without any setup.
2. Run the file `Downloader_V1.0.0-beta.exe` and follow the prompts.

### Logs and Download Directory
All downloads are saved in the `media` folder, and `download_history.csv` logs each download attempt with details such as URL, timestamp, and status.
